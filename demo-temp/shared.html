radio_button_checked buttons

Виджет выбора. Представлен всплывающим окном с набором кнопок.
  • remove_red_eye Пример виджета

Установка

Добавьте сценарии и таблицы стилей необходимые для работы библиотеки:
jQuery: 
                            <script src="https://code.jquery.com/jquery-3.4.1.min.js"></script>
kendo: 
                            <script src="https://kendo.cdn.telerik.com/2019.2.514/js/kendo.all.min.js"></script>
библиотека виджетов: 
                            <script src="https://rc-cdn/js/widgets.min.js"></script>
                        
                            <link href="https://rc-cdn/js/rcWidgets.css" rel="stylesheet"/>
Используйте глобальный объект rcWidgets для создания виджетов: 
                            
                            rcWidgets.buttons({
                            selector: '.buttons-demo-container-1',
                            buttons: [
                                {
                                    name: 'Отменить всё что можно',
                                    slug: 'cancel',
                                }
                            ],
                            title: 'Изменить статус',
                            fields: [{
                                name: 'Отменить 1',
                                slug: '1',
                            }, {
                                name: 'Отменить 2',
                                slug: '2',
                            }]
                        });

Конфигурационные параметры

Задаются объектом, свойства которого определяют внешний вид и поведение виджета.

selector

Определяет CSS-селектор для корневого элемента виджета.
  • remove_red_eye Пример
  • code Пример кода 
                        
                            rcWidgets.buttons({
                            selector: '.buttons-demo-container-1',
                            buttons: [
                                {
                                    name: 'Отменить всё что можно',
                                    slug: 'cancel',
                                }
                            ],
                            title: 'Изменить статус',
                            fields: [{
                                name: 'Отменить 1',
                                slug: '1',
                            }, {
                                name: 'Отменить 2',
                                slug: '2',
                            }]
                        });

buttons

Определяет коллекцию кнопок в нижней части виджета.
  • remove_red_eye Пример
  • code Пример кода

                            selector: '.buttons-demo-container-1',
                            buttons: [
                                {
                                    name: 'КАСТОМНАЯ КНОПКА ВНИЗУ',
                                    slug: 'cancel',
                                }
                            ],
                            title: 'Изменить статус',
                            fields: [{
                                name: 'Отменить 1',
                                slug: '1',
                            }, {
                                name: 'Отменить 2',
                                slug: '2',
                            }]
                        });

title

Определяет основной заголовок виджета.
  • remove_red_eye Пример
  • code Пример кода 
                        
                            rcWidgets.buttons({

                            buttons: [

                                    name: 'Отменить все',

                                name: 'Отменить 1',
                                slug: '1',
                            }, {
                                name: 'Отменить 2',

                            }]
                        });

                                name: 'ОСНОВНАЯ КНОПКА 2',

isDark

Определяет использование темной цветовой схемы.

                                }

                                name: 'Отменить 1',
                                slug: '1',
                            }, {
                                name: 'Отменить 2',
                                slug: '2',

                        });

pageX и pageY

Определяют положение появления виджета на экране.
  • remove_red_eye Пример
  • code Пример кода 
                        
                            document.getElementById('#myButton').addEventListener('click', function (e) {
                            rcButtons({
                            selector: '.my-container',
                            buttons: [
                                        {
                                            name: 'Отменить всё что можно',
                                            slug: 'cancel',
                                        }
                                    ],
                                    title: 'Изменить статус',
                                    fields: [{
                                        name: 'Отменить 1',
                                        slug: '1',
                                    }, {
                                        name: 'Отменить 2',
                                        slug: '2',
                                    }]
                            pageX: e.pageX,
                            pageY: e.pageY,);
                            });
                        });

Обработчики событий

Задаются в виде функции обратного вызова переданной в качестве втрого фактического параметра конструктора виджета.

обработчик закрытия окна

Пользовательский обработчик закрытия окна выбора. При выборе значения в качестве фактического параметра получает идентификатор (slug) такового.
  • remove_red_eye Пример
  • code Пример кода 
                        
                            rcWidgets.buttons({
                            selector: '.buttons-demo-container-7',
                                    buttons: [
                                        {
                                            name: 'Отменить всё что можно',
                                            slug: 'cancel',
                                        }
                                    ],
                                    title: 'Изменить статус',
                                    fields: [{
                                        name: 'Отменить 1',
                                        slug: '1',
                                    }, {
                                        name: 'Отменить 2',
                                        slug: '2',
                                    }]
                        }, function (v) {
                                    alert('Выбрано значение: ' + v);
                                });
