radio_button_checkedgetting started

Before You begin to use widgets, You have to add dependencies and make widgets library initialization.
As a dependencies, we use most recent versions of jQuery and kendo-ui libraries: Getting started with kendo-ui Getting started with jQuery
To use the library, reference the files in the script and link tags directly from the rc-widgets CDN domain. 
                            <script src="https://rc-cdn/js/widgets.min.js"></script>
                        
                            <link href="https://rc-cdn/js/rcWidgets.css" rel="stylesheet"/>
After appropriate dependencies install You have to run the widgets library initialization. Where selector is DOM-selector of the block which library going to use as widgets container. And the log is logging override by Your possible logging function implementation. 
                            
window.rc = rcWidgets({
    selector: '#widgets',
    log: function(message) {
        alert(message);
    }
});

After that You are ready to run the widgets by calling constructors: 
                            
window.rc.buttons({
        pageX: e.pageX,
        pageY: e.pageY,
        buttons: [
            {
                name: 'Cancel',
                slug: 'cancel',
            }
        ],
        title: 'Buttons Widget',
        fields: [{
            name: 'Selection 1',
            slug: '1',
        }, {
            name: 'Selection 2',
            slug: '2',
        }, {
            name: 'Selection 3',
            slug: '3',
        }],
    }, function(value) {
        alert('Your selection is: ' + value);
    });

                name: 'Cancel',

        fields: [{
            name: 'Selection 1',
            slug: '1',

            slug: '2',
        }, {
            name: 'Selection 3',
            slug: '3',

Configuration

pageX and pageY

Set the page coordinates where widget appears to be displayed.
                                
    document.getElementById('example-button')
    .addEventListener('click', function (e) {
        window.rc.buttons({
            pageX: e.pageX,
            pageY: e.pageY,
        });
    })
